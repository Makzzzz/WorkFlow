# WorkFlow

Привет! Это наш проект — WorkFlow.

## Что это?

Это веб-приложение для учебных групп, где можно создавать задания, отправлять решения и получать обратную связь от эксперта/одногруппников. Всё в одном месте, без лишних каналов и путаницы.

## Зачем?

Когда учишься в группе, часто бывает, что задания сдаются, но обратная связь теряется в бесконечных чатах и почте. Мы сделали это проще — все в одном месте, всё прозрачно и удобно.

## Как это работает?

1. Преподаватель создаёт задание
2. Студенты отправляют свои решения
3. Эксперт проверяет их и дает обратную связь или включает режим peer-ревью
4. Система автоматически распределяет работы для peer-ревью (анонимно)
5. Ты оцениваешь работу другого студента по критериям
6. Получаешь обратную связь от других студентов

## Что можно делать?

- Рисовать и выделять на картинках
- Использовать шаблоны комментариев
- Говорить, а не печатать — с помощью голосового ввода
- Создавать группы и приглашать одногруппников

## Для кого?

Для студентов и преподавателей, которые хотят упростить процесс сдачи и проверки заданий. Особенно полезно в групповых проектах и при выполнении практических заданий.

## Технологии

- Бэкенд: Python + FastAPI
- Фронтенд: React
- База данных: PostgreSQL
- Распознавание речи: Sber Salute Speech
